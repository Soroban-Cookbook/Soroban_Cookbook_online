# Testimonials Component

The `Testimonials` component renders a responsive community section for the homepage with:

- Developer testimonial cards
- Avatar image with initials fallback
- Optional social handle links per testimonial
- Community channel links (Discord, GitHub, Twitter/X, Telegram)

## Basic Usage

```tsx
import Testimonials from '@site/src/components/UI/Testimonials';

export default function HomeCommunitySection() {
  return <Testimonials />;
}
```

## Example (Default Data)

<Testimonials />

## Props

```ts
type Testimonial = {
  name: string;
  role: string;
  quote: string;
  avatarUrl?: string;
  socialHandle?: string;
  socialUrl?: string;
};

type CommunityLink = {
  label: string;
  url: string;
  icon: ReactNode;
  description: string;
};

type TestimonialsProps = {
  testimonials?: Testimonial[];
  communityLinks?: CommunityLink[];
  className?: string;
};
```

## Accessibility Notes

- Avatar images include descriptive `alt` text.
- External links use `target="_blank"` with `rel="noopener noreferrer"`.
- Community links and profile links include explicit `aria-label` values.
- Uses semantic `section`, `article`, `header`, and `blockquote` structure.
